Variables en javaScript
Una variable es la forma de almacenar un valor, para después poder utilizarlo. 
En javaScript existe tres formas de declarar una variable:

var nombre = 'Lidices';

let apellidos = 'Reyes Hung';

const apodo = 'Lidia';

var - Es es una palabra clave genérica para “variable”.
let - esta asignara una variable al igual que var, lo que difiere al crear un “nivel de scope”.
const- es una variable que no se podra cambiar.

Tambien podemos hablar de console.log(); que es un método que imprime lo que esta dentro del paréntisis.
